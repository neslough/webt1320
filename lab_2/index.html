Some Basic HTML Tags

<p> tags are used for generic text in paragraphs.
<ul> tags are used create bulleted lists:

  • Inside of an unordered list, <li> tags are used to represent individual items. Those items can have other tags within them.

  • <h3> is the third size of heading, and is useful for nested layers of titles, among other things.

  • <h6> is the smallest of the headings, and by default is actually smaller than the p tag.
  • Links to other pages can be created with the <a> tag.
